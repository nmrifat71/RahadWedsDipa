# 💍 Rahad Weds Dipa

A beautiful and responsive wedding invitation website created to celebrate the wedding of **Rahad & Dipa**.

## ✨ Features

- 💖 Elegant wedding invitation design
- 📱 Fully responsive for mobile and desktop
- ⏳ Wedding countdown timer
- 🖼️ Photo gallery
- 📍 Event details
- 🎵 Background music (if enabled)
- 💌 Smooth animations and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Live Demo

https://nmrifat71.github.io/RahadWedsDipa

## 📂 Installation

1. Clone this repository

```bash
git clone https://github.com/nmrifat71/RahadWedsDipa.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
